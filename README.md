# Attribute Convention for Data Discovery (ACDD)

The Attribute Convention for Dataset Discovery identifies and defines a list of NetCDF global attributes recommended for describing a NetCDF dataset to discovery systems such as Digital Libraries.

## Objectives

The ACDD is intended

- To allow efficient dataset discovery
- To allow users to use data efficiently
- To facilitate mapping between dataset metadata (notably netCDF) and ISO 19115.

## Current Version of ACDD

To view the current version of the ACDD, go to https://esipfed.github.io/acdd/.
